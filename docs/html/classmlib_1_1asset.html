Detailed Description

Class for storing asset files in Windows resource data.

This class provides a nice mechanism for 'hiding' the different assets needed by a HTTP server (pages, CSS files, images, etc.) inside the EXE file.

Each asset is stored as a user-defined resource of type RESFILE (defined as 256) and is identified by its ID. This function writes the asset to a file.

The resource file (.rc) should contain some lines like these: IDR_INDEX_HTML RESFILE "index.html" IDR_MAIN_CSS RESFILE "main.css"
