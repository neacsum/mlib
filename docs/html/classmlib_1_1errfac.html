Detailed Description

An error facility routes a group of errors handled in a similar manner.

To facilitate handling of ercs, each erc has associated a "facility". Instead of throwing an exception directly the erc calls the facility's raise function. In turn, it is this function that decides what should happen with the erc based on facility's log level and throw level.

There is also a default facility that is used when the erc doesn't have an explicit facility.
